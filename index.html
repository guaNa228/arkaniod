Hotkeys

SPACE - Spawn the ball/Start game

R - Restart game

SHIFT - Pause/Resume game

ESC - Toggle manual

W/S - Increase/Decrease number of rows

A/D - Decrease/Increase ball acceleration

/ - Move the paddle

Rules

The game globally has 2 states: gaming and setting

Gaming state is when you can see the ball on the field

And the setting state is when there is no ball on the playing field

Also gaming state is divided onto 2 substates active and paused

Active state is when the ball moves, paused state is when the ball stands still

During paused state you can just resume the game

During active state you can pause the game and move the paddle

To enter setting state from gaming state you can restart the game, or increase or decrease the number of rows or ball acceleration

During setting state you can increase or decrease the number of rows or ball acceleration
